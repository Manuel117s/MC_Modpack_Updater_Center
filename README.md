# 🎮 MC Modpack Updater Center

**MC Modpack Updater Center** es una plataforma centralizada diseñada para simplificar la gestión, publicación y seguimiento de actualizaciones para modpacks de Minecraft. Nuestra herramienta permite a los creadores de contenido y sus equipos mantener a su comunidad informada con un historial claro de versiones y noticias.

## ✨ ¿Qué puedes hacer en la App?

Como usuario de esta plataforma, tendrás el control total sobre tus paquetes de mods:

* **Dashboard Personalizado:** Visualiza rápidamente todos los modpacks que posees o en los que colaboras, junto con la versión más reciente de cada uno.
* **Gestión de Versiones:** Publica nuevas actualizaciones y parches de manera organizada. La app actualiza automáticamente la referencia de la "versión más reciente" para que tus usuarios siempre sepan qué descargar.
* **Comunicación Directa:** Publica noticias y anuncios relevantes dentro de cada modpack para mantener a tu comunidad al tanto de cambios, nuevas funciones o correcciones.
* **Trabajo en Equipo:** ¿Tienes colaboradores? Puedes asignar roles (Administrador, Editor, Visor) para que tu equipo te ayude a mantener el modpack actualizado sin que tú tengas que hacerlo todo solo.
* **Visibilidad Flexible:** Define si tus modpacks serán privados (solo para tu equipo) o públicos (para que cualquier persona pueda ver el historial y las novedades).

## 🚀 Cómo empezar

1.  **Registro / Login:** Crea una cuenta en la plataforma para acceder a tu panel de control.
2.  **Crea tu primer Modpack:** Haz clic en el botón de "Create Modpack", define un nombre, versión del juego y nivel de visibilidad.
3.  **Gestiona tu equipo:** Invita a otros usuarios a colaborar en tu proyecto desde la sección de configuración de tu modpack.
4.  **Lanza actualizaciones:** Crea nuevas versiones y publica noticias para empezar a construir tu historial de cambios.

## 🛠️ Tecnologías utilizadas

* **Frontend:** Desarrollado con React para una interfaz rápida, intuitiva y dinámica.
* **Backend:** Construido con FastAPI para una respuesta eficiente y segura a todas las solicitudes de datos.
* **Base de Datos:** MongoDB, garantizando flexibilidad para almacenar el historial de versiones y noticias.
