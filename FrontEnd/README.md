# MC Modpack Updater Center

MC Modpack Updater Center is a place to keep Minecraft modpacks organized and up to date.

It is made for people who want to:

- keep track of their modpacks in one place
- see the latest releases and updates
- share news about changes with players or team members
- work with other people on the same modpack
- manage access to each modpack in a simple way

The goal of the project is to make modpack management easier, so you spend less time looking for updates and more time playing and sharing them.

## What you can do

- Sign in and open your modpack space
- Check the latest version of each modpack
- Add or update information about a modpack
- Follow news and release updates
- Work together with collaborators

## Who this is for

This app is for modpack creators, server owners, and teams who want a simple way to keep everything organized.

## In short

If you manage Minecraft modpacks, this project helps you keep them clear, current, and easy to share.

## Environment

Set the frontend API base URL before running or building the app when the backend is hosted separately:

- `VITE_API_BASE_URL=https://api.your-domain.com`

If the frontend and backend are served from the same origin, you can leave this empty and the app will call relative API paths.
